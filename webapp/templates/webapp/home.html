About DevOps

DevOps is a set of practices that combines software development (Dev) and IT operations (Ops). It aims to shorten the development lifecycle and provide continuous delivery with high software quality. Here are some tools that I work with:

Jenkins

Automation server to build, test, and deploy applications efficiently.

Docker

Platform to build, run, and share containerized applications.

Kubernetes

Container orchestration platform to manage and scale applications.

AWS

Cloud platform offering reliable and scalable infrastructure.

Ansible

Configuration management tool for automating IT tasks.
